# 🗡️ Zelda Mini Adventure

A complete top-down Zelda-style adventure game built with Pygame.

## Features
- Smooth 4-directional movement (WASD / Arrow keys)
- Sword combat — press **SPACE** to attack in the direction you're facing
- 5 aggressive slimes that chase you
- Collectible rupees (green diamonds)
- Health system with hearts + invincibility frames
- Knockback on hits
- Stone pillars to navigate around
- Win / Lose screens with restart (press **R**)

## How to Play

```bash
git clone https://github.com/Bazzax/zelda-mini-adventure.git
cd zelda-mini-adventure
pip install -r requirements.txt
python3 main.py
```

### Controls
| Key          | Action          |
|--------------|-----------------|
| WASD / Arrows| Move            |
| **SPACE**    | Swing sword     |
| **R**        | Restart game    |

## Requirements
- Python 3.8+
- Pygame 2.6+

## Controls
- Move with **WASD** or **Arrow Keys**
- Attack with **SPACEBAR**
- Restart with **R** after winning or losing
